# 管理面板说明

Bot 启动后，管理员和 owner 会看到更多命令与按钮。常用入口有两个：

- `/start`: 用户面板，管理员会多出管理入口。
- `/config`: 高级配置面板，主要给 owner 和管理员调整运行参数。

## 注册管理

注册管理用于控制用户创建 Emby 账号的方式。

| 功能 | 说明 |
| --- | --- |
| 自由注册 | 开启后，用户可直接在面板创建账号。 |
| 定时注册 | 按“持续时间 + 总人数”限时开放注册。 |
| 总人数限制 | 到达上限后自动停止注册。 |
| 生成注册码 | 生成注册或续期用的 code / 深链接。 |
| 查看注册码 | 查看管理员生成的注册码、使用状态和剩余数量。 |

生成注册码格式：

```text
天数 数量 模式 续期类型
```

示例：

```text
30 5 link F
```

含义：生成 5 个 30 天注册深链接。

模式说明：

- `link`: 深链接，用户点击后跳转 Bot。
- `code`: 普通兑换码。
- `F`: 注册码。
- `T`: 续期码。

## 配置面板

`/config` 面板用于在线修改部分 `config.json` 参数。

| 按钮 | 作用 |
| --- | --- |
| 导出日志 | 发送当天 `log/log_日期.txt`。 |
| 设置探针 | 设置 Nezha V0、Nezha V1 或 Komari。 |
| Emby 线路 | 修改普通用户看到的线路说明。 |
| 白名单线路 | 修改白名单用户专属线路说明。 |
| 显隐媒体库 | 设置用户可自行显示/隐藏的媒体库。 |
| 注册码续期 | 控制用户能否使用注册码续期。 |
| 退群封禁 | 用户退群后是否封禁。 |
| 播放结算 | 播放榜结算时是否给用户积分。 |
| 自动更新 | 是否每日自动拉取 Bot 代码。 |
| MoviePilot | 配置点播功能。 |
| 分区通行码 | 管理分区媒体库授权码。 |
| 红包设置 | 开关红包和专属红包。 |

!!!tip

    面板修改会写回 `config.json`。如果手动改配置文件，建议重启 Bot 后再进入面板确认状态。

## 定时任务面板

定时任务受 `schedall` 配置控制。

| 任务 | 默认用途 |
| --- | --- |
| `dayrank` | 每日媒体播放次数榜。 |
| `weekrank` | 每周媒体播放次数榜。 |
| `dayplayrank` | 每日用户观看时长榜。 |
| `weekplayrank` | 每周用户观看时长榜。 |
| `check_ex` | 到期检测，过期后封禁，超过冻结天数后删除。 |
| `low_activity` | 活跃检测，长时间未活跃则处理。 |
| `partition_check` | 分区通行码到期检查。 |
| `backup_db` | 数据库定时备份。 |

## 用户管理

常见用户管理能力：

- 查询用户信息。
- 修改积分或花币。
- 调整到期时间。
- 加入或移除白名单。
- 删除 Emby 账号。
- 删除 Bot 数据库记录。
- 管理额外媒体库权限。
- 查询用户设备、IP、客户端记录。

删除类操作建议先执行 [备份与恢复](../deploy/maintenance.md)。

## 权限层级

| 等级 | 说明 |
| --- | --- |
| owner | `config.json` 中的 `owner`，拥有最高权限。 |
| admin | `admins` 中配置的管理员，或 owner 通过命令添加。 |
| 白名单用户 | Bot 内用户等级为 `a`。 |
| 普通用户 | Bot 内用户等级为 `b`。 |
| 封禁用户 | Bot 内用户等级为 `c`。 |
| 未注册用户 | 无有效 Emby 账号记录。 |
